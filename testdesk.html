Common Terms & Applications

  • Remote Desktop Connection: These are connections like TeamViewer and AnyDesk that allow a user to access their workstation remotely
  • Network Printer: Most of the Department's printers are in this category. These are printers that can be used from anywhere within the HPD Network and do not require a physical connection
  • Softphone: Sofphones are software that allow phone calls to be made via the internet. These are applications like Cisco Jabber
  • Server: Servers are computers that host the data and applicaitons that we use every day. Some our our servers host things like RMS and CAD
  • VPN: VPN means "Virtual Private Network." VPN's allow a user to securely access a network that you are not physically connected to. An example of this is NetMotion, which we use to access the HPD network in patrol cars.
